ABOUT

This API assignment was created by Edwin Christie and Michael Valadao-Martins. The API we decided to create was a file share for users to be able to upload a file and then allow other users to download the file when given a custom link. The file availability default will be set to one hour unless specified different by the user. Once the file is no longer available it will be deleted from the server. We have also given the option to make the uploaded files private for users who are logged in.
